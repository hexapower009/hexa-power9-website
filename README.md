# Hexa Power 9 Website

موقع Landing Website احترافي لفرع Hexa Power 9 - حي النخيل، الرياض.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- Localized Arabic/English routes: `/ar` و `/en`
- Ready for Vercel deployment

## Run locally

```bash
npm install
npm run dev
```

افتح:

```txt
http://localhost:3000/ar
http://localhost:3000/en
```

## Deploy on Vercel

1. ارفع الملفات على GitHub.
2. افتح Vercel.
3. Import Project من GitHub.
4. Deploy.
5. بعد شراء الدومين، غيّر `example.com` داخل:
   - `app/sitemap.ts`
   - `components/JsonLd.tsx`

## Important Notes

- الأسعار غير مثبتة داخل الموقع، حتى يتم ضبط خطة التسعير لاحقًا.
- كل أزرار واتساب موجهة إلى فرع النخيل فقط.
- الصور مضغوطة WebP لتحسين سرعة الموقع.
- اللوجو المستخدم حاليًا نسخة مقصوصة على خلفية سوداء مناسبة للموقع، ويُفضّل لاحقًا توفير PNG شفاف حقيقي أو SVG.
